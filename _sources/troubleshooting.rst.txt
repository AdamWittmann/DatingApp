Troubleshooting
===============
This section describes some common issues that can arise and possible solutions.

Issue #1
--------
Can't connect to database

make sure you have created a .env file with your database name, owner and password

Issue #2
--------
links between pages won't work

make sure your webapges are set up in app.py

Issue #3
--------
PgAdmin won't work

make sure you only have one version of Sqlalchemy running at a time
